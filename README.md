# Olfactory Masterpiece

This is a premium, highly responsive single-page web application replicating the "Our Story" timeline of a luxury perfume brand.

## Features
- Pixel-perfect implementation of a design mockup
- Highly customized, premium typography using Google Fonts (Anton and Montserrat)
- Fully responsive design that maintains the desktop layout proportions on mobile devices
- Complex flexbox alignments with overlapping absolute-positioned SVG and image elements
- Custom underline styles and styling adjustments

## Technologies
- HTML5
- CSS3 (Flexbox, Media Queries, Custom Properties)
